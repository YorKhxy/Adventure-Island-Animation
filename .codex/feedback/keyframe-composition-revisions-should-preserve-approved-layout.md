# 关键图构图修订应保留已认可布局

## Context

用户要求重做 SEG01 关键图时指出，原图的问题只是镜头太近、冒险岛场景展示不足；后续生成把人物位置和整体构图改得太多，偏离了用户认可的原布局。

## Lesson

当用户认可一张关键图的环境和人物布局，只指出“拉远一点”“多展示场景”等局部构图问题时，不要重新发明画面。应保留角色位置、动作关系、场景主体和叙事功能，只做轻微镜头距离、留白或景深调整。

## Rule

- 先判断用户是在否定整张图，还是只要求局部构图调整。
- 如果用户说“整体还是按这张来”，必须把原图作为构图锁定参考。
- “拉远一点”通常是 10%-15% 的轻微 zoom out，不是重做远景建立镜头。
- 修订后要检查角色动作关系是否仍匹配原提示词和可灵分镜。
