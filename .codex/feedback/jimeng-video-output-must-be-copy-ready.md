# 即梦视频交付必须直接给可复制操作和提示词

## Context

用户要求创建针对即梦生成视频的 Skill，并明确每次说“去即梦导入”或“用即梦生成视频”时，必须把即梦需要的内容、操作方式、提示词全部列出来，不能贴文档地址让用户自己看。

## Lesson

视频生成平台的交付物是可执行输入，不是资料索引。用户需要边看界面边复制粘贴，所以必须按平台界面顺序输出基础设置、上传素材、素材用途、主提示词、负面提示词、台词/字幕、检查标准和保存命名。

## Rule

- 命中即梦/Jimeng/即梦AI/去即梦导入/用即梦生成视频时，必须先读取 `.codex/skills/jimeng-video-builder/SKILL.md`。
- 输出不能只贴文档地址、文件路径或行号。
- 所有需要粘贴到即梦的提示词必须放进可复制代码块。
- 即梦和可灵能力边界不同，不能把可灵主体绑定、音画同步规则无脑套给即梦。
