# 参考素材用途必须明确分工

## Context

用户要求继续查公开视频生成提示词参考。重新联网后抓到 Runway / Seedance 2.0 和 Runway API recipe 文档：Seedance 指出 Reference mode 需要在提示词里说明每个输入如何使用；Runway reference media guidelines 指出参考素材要高质量、主体清楚、用途单一；Product Ad recipe 也强调用创意方向描述场景、灯光、镜头运动，而让参考图定义外观。

## Lesson

在可灵、Runway、Seedance 这类视频生成工具中，参考图不能模糊写成“都参考”。必须明确每个素材的职责：首帧负责构图和初始位置，主体参考负责角色身份，风格参考负责色彩和光影，动作/分镜文本负责运动。否则模型会把参考素材互相混用，导致角色漂移、动作混乱或风格不一致。

## Rule

- 每次上传参考图后，提示词必须写清“这个素材用来控制什么，不控制什么”。
- 首帧图：控制第一帧、构图、场景、初始位置。
- 主体参考：控制指定角色身份、脸、发型、服装，不控制全局构图。
- 未上传/未绑定的角色参考，不要在提示词里写成已绑定主体。
- 创意方向负责写场景、灯光、镜头运动和叙事，不要让参考图承担所有任务。
