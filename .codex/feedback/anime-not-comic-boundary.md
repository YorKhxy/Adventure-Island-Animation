# 动漫短剧边界不要降级成漫画短剧

**问题描述**：主 Agent 将用户的“AI漫画短剧带解说”延展为“漫画分镜/漫画短剧”，用户纠正“我要的是动漫不单单是漫画”。

**触发场景**：产品需求收集阶段，讨论小说改编短视频工作流时。

**教训/建议**：后续应使用“动漫短剧”作为产品目标，覆盖动态镜头、动作、转场、配音、字幕和最终视频预览，不要只围绕静态漫画图或漫画分镜设计。
