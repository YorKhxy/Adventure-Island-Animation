# 儿童原创改编不要凭空添加角色名

**问题描述**：主 Agent 根据用户一句“主角是浩浩”直接确认主角叫浩浩，随后用户纠正“主角是沐沐，没有浩浩”。

**触发场景**：儿童原创小说《初入冒险岛》角色连续性设定阶段。

**教训/建议**：儿童手稿字迹和人名容易误读。角色姓名、主角身份、人物关系必须要求用户确认，不能把未出现在原稿中的名字写进连续性设定。
