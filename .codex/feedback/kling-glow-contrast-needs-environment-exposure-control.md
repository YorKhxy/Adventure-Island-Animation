# 可灵发光效果不足要控制环境曝光对比

## Context

用户在 SEG01 可灵视频生成后反馈：岛门最后没有明显亮起来。进一步修正：不是只把门写得更亮，因为当前环境本身已经很明亮，门光和环境对比不够强；应当适当把清晨环境调暗一点点，但不能调得太过，要保留清晨感觉。

## Lesson

在可灵视频提示词里处理“发光物需要明显亮起”时，不能只堆“更亮、更强光”。如果初始环境亮度过高，模型会把发光物和背景一起做亮，导致亮起变化不明显。正确做法是同时控制环境曝光基准：轻微压低草地、树影和远景背景亮度，保留天空清晨感，再让发光物在后半段增强轮廓光、外溢光晕和边缘受光。

## Rule

- 写发光门、魔法入口、能量核心等亮起效果时，必须同时写环境对比关系。
- 清晨场景可用“略低曝光清晨光效”“草地和树影比首帧略暗5%-10%”“天空仍清澈明亮”来建立对比。
- 禁止只写“更亮、更强光”，也禁止把环境压成夜晚、阴天或恐怖黑暗。
- 通过标准要包含：发光物明显亮起、边缘光/外溢光晕可见、周围材质有受光、整体仍符合原场景时段。
