# 角色能力职能不能互相抢招数

**问题描述**：钰钰原设定使用了“金盾/护盾”，用户修正说盾是浩浩的招数，钰钰虽然是金属性，但不能使用盾。同时用户要求所有人物五官、发型、服装配色都要明显不同，尤其钰钰金属性和沐沐光属性配色接近时必须一眼区分。后续用户又指出钰钰琥珀黄/深金/橙棕服装和浩浩棕黄土系太像，因此相近暖色系也会造成角色混淆。

**触发场景**：角色设定图生成流程中，AI 按旧提示词生成了包含 shield/golden shield 概念的钰钰候选，导致钰钰能力职能和浩浩保护/防御职能重叠，也存在和沐沐白金光效混淆风险。随后改成金线/金铃后，服装仍使用琥珀黄、深金、橙棕，又与浩浩棕黄土系撞色。

**教训/建议**：为多角色团队生成设定前，先建立能力职能边界：谁负责防御、谁负责治疗、谁负责控场、谁负责攻击/辅助。相近属性角色必须先写硬区分锚点，包括五官气质、发型、服装轮廓、主配色面积、魔法形态。生成提示词中要明确禁止抢占其他角色核心招数，也要禁止使用已被其他角色占用的主色系；金属性不等于服装必须大面积金黄，可以用冷色服装加金属点缀保留属性。
