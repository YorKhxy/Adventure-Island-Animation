# 可灵导入说明必须按界面顺序给可复制文本

## Context

用户要求：每次问“怎么导入可灵制作”时，不要只给文件路径和行号，而是把需要填写的提示词按可灵界面顺序列好，方便直接复制粘贴。

## Lesson

可灵制作交付不是文档索引任务，而是操作执行任务。用户站在可灵界面前，需要的是“上传什么、哪个框填什么、按什么顺序复制”的清单。行号引用只能作为备份，不应作为主要交付形式。

## Rule

- 用户问“怎么导入可灵 / 怎么填可灵 / 可灵制作怎么操作”时，必须按界面顺序输出：基础设置、上传素材、主体绑定、主提示词、自定义分镜、负面提示词、音画同步/台词、音色、保存命名。
- 需要填写的每一段提示词必须直接给可复制文本块。
- 文件路径和行号只作为辅助来源，不代替复制文本。
- 如果有自定义分镜，必须按分镜编号分别给文本，不让用户自己从长文档里拆。
- 即使已经把导入执行区写进项目文件，最终回复仍必须直接贴出可复制交付内容；不能只说“已写入某文件”。用户站在可灵界面前，缺少复制文本就等于没有交付。
- 可灵界面默认只显示 1-2 个镜头框时，不代表只能按默认数量写。只要界面支持“添加镜头”，必须按视频效果最合理的镜头数设计分镜；复杂动漫段落优先 3-5 镜头，不能为了迎合默认框数把节奏压扁。
- 如果当前可灵界面没有独立负面提示词框，就不要交付“负面提示词框填这个”。必须把禁止项拆进各个镜头文本末尾，用“禁止：...”短句表达，并按镜头风险分配，避免用户无处可贴。
- 如果用户截图显示自定义分镜界面只有镜头文本框，没有全局提示词框，就不要再输出“全局提示词”。必须把全局风格、角色一致性、时间逻辑、负面约束全部拆进每个镜头文本，让用户按镜头1/2/3/4直接复制粘贴。
- 可灵自定义分镜的每镜头秒数可以调整时，不要照抄界面默认 `3s/4s/4s/4s`。必须按剧情节拍反推秒数，例如需要 6 秒出现转折、10 秒进入高潮时，应安排 `3s/3s/4s/5s` 这类更贴合事件节点的时长。
