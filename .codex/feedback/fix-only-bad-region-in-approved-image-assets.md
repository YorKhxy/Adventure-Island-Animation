# 图片资产局部瑕疵只修问题区域

**问题描述**：用户认可方方动作表整体效果，只指出左下跑步动作出现三只脚。主 Agent 先把整张动作表重做，导致原本好的动作也被替换，属于过度修复。

**触发场景**：角色动作表、表情表、分镜图、关键帧等已被用户认可的图片资产，只存在局部 AI 崩坏、肢体错误、文字水印或小范围瑕疵时。

**教训/建议**：默认先做局部修复：保留用户认可原图，归档当前异常版本，制作遮罩或局部编辑区，只重绘有问题区域；必要时再用本地合成保证遮罩外像素不变。除非用户明确要求整张重做，否则不要用全图重生成替代局部修复。
