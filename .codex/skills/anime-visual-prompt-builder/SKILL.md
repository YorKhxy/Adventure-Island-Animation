name: anime-visual-prompt-builder
description: 当动漫短剧分镜需要转成 ComfyUI、Stable Diffusion、SDXL、FLUX 或 AI 视频生成提示词时使用。产出角色一致的图片/视频提示词表。
---

[任务]
    将动态分镜和连续性设定转成可用于 ComfyUI、Stable Diffusion/SDXL/FLUX 和 AI 视频工具的提示词、负面提示词和参考图策略。

[依赖检测]
    Skill 启动时第一步自动执行。

    必需：
    - anime-shot-designer 产出的分镜表。

    可选：
    - anime-continuity-bible 产出的角色固定提示词。
    - 用户已有 ComfyUI 工作流、模型名称、LoRA、参考图。

    降级策略：
    - 无模型信息时输出通用提示词，不写死具体节点参数。
    - 无参考图时加强文字锚点和负面提示词。

[第一性原则]
    **角色一致性优先**：每个镜头复用角色锚点，不为单镜头好看牺牲连续性。
    **提示词分层**：角色、场景、动作、镜头、画风、光影分开写，方便替换。
    **少写玄学词**：不堆无意义形容词，提示词必须服务画面。
    **生成路线分离**：图片提示词和视频提示词不能混用。
    **联网优先**：具体模型授权、参数建议、工具变化不确定时先搜索确认。

[文件结构]
    ```
    anime-visual-prompt-builder/
    └── SKILL.md
    ```

[视觉提示词清单]
    每个镜头必须包含：
    - 镜头编号
    - 角色锚点
    - 场景锚点
    - 动作描述
    - 景别和构图
    - 镜头运动提示
    - 光影和色彩
    - 正向提示词
    - 负向提示词
    - 一致性备注
    - 推荐生成方式

    角色参考必须包含：
    - 固定外观短词
    - 服装锚点
    - 标志道具
    - 禁止变化项

[工作流程]
    [第一步：读取连续性锚点]
        先提取角色固定提示词和画风规则，没有就标记待补。

    [第二步：逐镜头生成提示词]
        按镜头表输出图片提示词和必要的视频提示词。

    [第三步：生成负面提示词]
        针对角色漂移、肢体错误、现代穿帮、字幕水印等生成负面提示词。

    [第四步：标记制作风险]
        标出难生成镜头、需要参考图镜头、适合改成静态运镜的镜头。

[输出格式]
    - **全局画风提示词**
    - **角色固定提示词**
    - **镜头提示词表**
    - **负面提示词库**
    - **ComfyUI/本地生成备注**
    - **AI 视频片段提示词候选**
    - **需要人工补素材项**

[初始化]
    执行 [第一步：读取连续性锚点]。
