# Mumu Production Notes

## Final Reference

- 主参考图：`character-mumu-final.png`
- 来源：最终版已选定，生成过程已归档。
- 用途：所有后续三视图、表情表、动作表、分镜、关键帧统一以此为最高优先级。

## Visual Anchors

- 年龄：7 岁，必须是儿童比例。
- 身份：浩浩的妹妹，有妹妹感，但关键时刻坚定。
- 发型：双丸子头，淡金色蝴蝶结。
- 服装：白色、淡金色小斗篷，柔亮童话国漫风，精致但不能继续公主化。
- 魔法：柔和治疗光球、暖金白光、保护光圈、轻柔粒子。

## Consistency Rules

- 不要画成成年少女、偶像魔法少女或过度华丽公主。
- 不要使用皇冠、婚纱、成人礼服。
- 不要出现十字、医疗十字、宗教符号或医院图标。
- 治疗光不能像攻击激光，不能做成战斗光束。
- 不添加武器、血腥、恐怖元素。
- 和浩浩同框时，沐沐应明显更小、更柔和，浩浩负责保护感，沐沐负责治疗和情感理解。
