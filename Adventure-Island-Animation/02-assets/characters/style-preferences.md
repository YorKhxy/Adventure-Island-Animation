# Character Style Preferences

## Current Preferred Style

用户明确喜欢当前沐沐 final 的柔亮童话国漫风格，并要求全系列角色统一到这个方向。

### 风格关键词

- 柔亮童话国漫风。
- 暖奶油色干净背景。
- 大而柔和的光圈/保护罩效果。
- 精致但仍儿童友好的服装细节。
- 圆润脸型、柔和高光、干净线条。
- 魔法效果明亮、温暖、治愈，不阴暗、不写实。

### 应用规则

- 后续角色主参考优先向 `mumu/character-mumu-final.png` 的风格统一。
- 六个孩子当前 final 和三件套均已统一到柔亮童话国漫风格。
- 每个角色仍必须保留自己的魔法属性差异，不能全部变成金色光效。
- 过程候选、旧参考、旧提示词已归档到 `../../98-archive/`，不得作为 active 生产输入。

### 风险

- 风格容易过度公主化，需要保持儿童冒险感。
- 光效容易混淆：浩浩土魔法应偏砂石色，沐沐治疗光偏暖白金，钰钰金属性应偏金属星片和金线，但服装主色必须用星夜蓝/靛紫来避开浩浩棕黄和沐沐白金。
- 钰钰必须和沐沐一眼区分：不同五官气质、不同发型、不同服装轮廓、不同配色面积；钰钰不使用双丸子头、白金斗篷或治疗光球。
- 钰钰不能使用盾，盾/防御墙属于浩浩的保护招数。
- 钰钰不能使用大面积棕黄、土黄、橙棕服装，否则会和浩浩撞色；金色只能作为星发卡、金线、星片、铃铛等点缀。
- 新会话继续时，直接使用各角色目录根部的 `character-*-final.png` 和无版本号三件套，不要去归档目录翻候选图。
