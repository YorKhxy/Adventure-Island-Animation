# Yuyu Production Notes

## Final Reference

- 主参考图：`character-yuyu-final.png`
- 来源：最终版已选定，生成过程已归档。
- 用途：所有后续三视图、表情表、动作表、分镜、关键帧统一以此为最高优先级。

## Visual Anchors

- 年龄：7 岁，必须是儿童比例，后续三视图要压低腿长。
- 性格：自信、可靠、外向，像主动帮忙的朋友。
- 发型：利落短发/bob，金色星星发卡。
- 服装：星夜蓝、靛紫、深蓝紫为主的冒险装，金色只做星发卡、装饰线、星片、铃铛等点缀；短外套、短裤/裙裤、深色靴子、小挎包。
- 魔法：金线、金属星片、金铃/星铃共鸣环、金色轨迹、标记光点。

## Consistency Rules

- 绝对不要生成盾、金盾、护盾、防御墙。
- 不要和沐沐撞：不用双丸子头、不穿白金小斗篷、不拿治疗光球、不大面积白色。
- 不要和浩浩撞：不做防御墙、不做保护盾，不用棕黄/土黄/橙棕作为服装主色，浩浩负责土墙和保护。
- 不要把服装改回琥珀黄、深金、橙棕主色；这些会撞浩浩。
- 金属性是“金线牵引、星片标记、铃声共鸣、辅助控场”，不是治疗，也不是盾防。
- 保持柔亮童话国漫风，但服装不能偶像化、成人化；金色不能铺满全身。
