## 可灵词库预设点击

自定义分镜是主方案。智能分镜只当降级。原因很简单：这段要先交代兄妹，再交代伙伴，最后交代门；一上来六个人全景站满画面，会死得很整齐。

如果界面只允许全局选择一套预设，优先这样点：

- 镜头：`环绕推进`。
- 基础镜头：`拉远` + `跟随`，不要选强烈手持。
- 运镜速度：`慢速`。
- 景别：`中景`，并在提示词里写清 0-3 秒近景、10-15 秒远景。
- 视角：`正面视角` + 少量 `侧面视角`。
- 光影：`阳光` + `自然光` + `柔和光` + `暖光`，不要城市夜景，不要强烛光。

如果自定义分镜可以逐镜头设置，按下面四段填：

- 0-3秒：`环境下摇` / `近景` / `景深` / `固定镜头` / `自然光` / `柔和光` / `慢速`。
- 3-6秒：`跟随` / `中景` / `侧面视角` / `柔和光` / `暖光` / `慢速`。
- 6-10秒：`环绕` / `拉远` / `中景` / `正面视角` / `自然光` / `慢速`。
- 10-15秒：`上升推进` / `拉远` / `远景` / `景深` / `阳光` / `暖光` / `慢速`。

不要贪心把所有镜头词都点满。全局选择太多会互相打架，结果又变成抽风镜头。宁可少，但顺。

## 中文正向提示词

使用最新关键图作为场景、角色和结尾远景构图锚点。保持关键图里的服装颜色、儿童比例、同一座藤蔓石拱潮汐洞门、右侧完整门光特效和深蓝黑洞道一致，但不要把第一秒做成六个孩子全景站桩。前半段允许用镜头裁切、景深虚化、前景草叶遮挡和跟随运镜，先只清楚交代浩浩、沐沐，再逐步揭示其他孩子。门洞必须是看不见尽头的深水潮汐洞道，不要出现明亮出口、天空、海平线、第二个山洞或第二个门。

整体目标：15秒国产院线级国漫儿童冒险开场，不是静态插画轻微晃动。参考国产院线动画的表演密度、弧线动作、镜头调度和情绪推进，但不要复制任何已有电影角色、服装、火焰造型或具体镜头。画面要有“醒来、保护、伙伴逐个进入叙事、入口苏醒”的清楚节奏。角色有呼吸、眨眼、重心转移、衣摆和发丝延迟；镜头有前后景视差、遮挡揭示和注意力引导。

画风：柔亮童话国漫风，清晨冒险岛入口，暖奶油阳光、自然光、柔和光。天空明亮，草地和树影略低曝光形成层次。植物保持干净青绿、嫩绿、墨绿色块，不要整片黄绿、脏金、过曝暖光。只允许少量微光点集中在门框附近，植物叶面、草地、藤蔓、水草上不要密集小黄点、小白点、假露珠、萤火粒子、随机闪光或芝麻状高光。

角色动作：浩浩坐在前景草地上，双腿和两只脚完整可见，先茫然眨眼，慢慢抬头，听到沐沐害怕后肩膀微微前压，身体挡在沐沐前面。沐沐抓紧浩浩衣角，轻轻缩肩，靠近哥哥，披肩和发丝被微风带动。钰钰在后方发现门框符文，迈半步抬手指向发光石拱。骆骆半蹲触碰浅水，出现干净蓝白同心涟漪。卉卉轻触藤叶，叶脉微微泛绿。方方在右侧朝门洞方向小跑两三步，动作有蓄势、迈步、停住，不穿过卉卉，不撞人。

镜头运动：按可灵词库预设执行慢速运镜。0-3秒用环境下摇和近景景深，从草叶、浅水反光和前景兄妹进入，只清楚交代浩浩与沐沐；3-6秒跟随浩浩和沐沐的视线，轻微侧面视角，后景伙伴仍可虚化；6-10秒环绕拉远到中景，通过遮挡揭示逐个带出钰钰、骆骆、卉卉、方方；10-15秒上升推进并拉远到远景，才完整看到六个孩子和同一座藤蔓石拱潮汐洞门。镜头必须稳定，有电影感，不要剧烈手持、突然变焦、无动机快切。

环境动态：草叶轻摆，水面有低频涟漪，门洞里吹出很轻的气流，孩子发丝、披风、衣摆有延迟摆动。藤蔓石拱门从左下和右下同时开始亮，金白符文沿左右两侧完整爬升，右侧不能缺光。门洞内部始终是深蓝黑岩洞水道，看不见尽头，只在深处有微弱蓝色层次，不要变成白色传送门。

## 可灵自定义分镜填入版

### 分镜1，0-3秒，环境下摇到前景兄妹

词库预设：`环境下摇`、`近景`、`景深`、`固定镜头`、`自然光`、`柔和光`、`慢速`。

镜头从草叶和浅水反光轻轻下摇到前景浩浩、沐沐。画面只清楚显示兄妹，其他孩子即使在关键图中存在，也要被景深、草叶和构图裁切压到背景，不要一上来全员清楚摆拍。浩浩坐在草地上，双腿双脚完整可见，茫然眨眼后慢慢抬头。沐沐抓紧浩浩衣角，害怕但不哭。前景草叶只做柔和视差，不要用满屏光点带节奏。

### 分镜2，3-6秒，跟随兄妹视线

词库预设：`跟随`、`中景`、`侧面视角`、`柔和光`、`暖光`、`慢速`。

镜头跟随浩浩和沐沐一起看向远处门洞。浩浩先看沐沐，再轻轻挡在她前面，表情从迷茫变坚定。沐沐手指收紧，身体靠近哥哥。远处藤蔓石拱潮汐洞门微微增强金白光，但洞内仍是看不见尽头的深蓝黑水道。

### 分镜3，6-10秒，环绕拉远带出伙伴

词库预设：`环绕`、`拉远`、`中景`、`正面视角`、`自然光`、`慢速`。

镜头慢速环绕并拉远，借前景草叶、浅水反光和角色身体遮挡，逐个揭示四个伙伴，不要四人同时清楚出场。钰钰先迈半步指向门框符文；骆骆随后半蹲触水，脚边出现蓝白同心涟漪；卉卉再轻触藤叶，绿色叶脉微光一闪；方方最后从右侧小跑靠近门洞方向。所有动作有预备、主动作、停顿和余韵。

### 分镜4，10-15秒，上升推进到入口苏醒

词库预设：`上升推进`、`拉远`、`远景`、`景深`、`阳光`、`暖光`、`慢速`。

镜头轻微上升推进后拉远，六个孩子同时看向同一座藤蔓石拱潮汐洞门。门框左右两侧符文完整亮起，金白光沿石拱爬升，右侧特效不能断。洞内深蓝黑岩洞水道看不见尽头，不出现出口。最后0.5秒停在门光最强但不过曝的远景，方便接 SEG02 能力觉醒。

## 音画同步脚本

建议开启音画同步，但字幕后期添加。若可灵必须生成字幕，只允许底部白字黑描边，不能遮挡人物。

- 00:00.5-00:05.0，旁白，温和、神秘：浩浩一醒来，就发现自己来到了陌生的冒险岛。
- 00:05.2-00:08.0，沐沐，轻声害怕但不哭：哥哥，我有点害怕。
- 00:08.2-00:11.0，浩浩，压住紧张、坚定：别怕，我会保护你。
- 00:11.2-00:14.8，旁白，期待感增强：远处，一座发光的潮汐洞门慢慢亮了起来。

口型只绑定沐沐和浩浩。旁白不绑定口型。钰钰、骆骆、卉卉、方方只做动作反应，不要开口。

## 生成风险

- 六人同屏容易变脸，主体参考优先锁浩浩和沐沐。
- 门洞右侧光效容易缺失，正向和负向都必须强调左右两侧完整点亮。
- 可灵可能把深洞画成明亮出口，必须反复写“看不见尽头”。
- 粒子和微光容易扩散到植物表面，必须限制在门框附近。
- 动作容易僵硬，所以每个角色必须写预备、主动作、停顿、余韵。
