# Project Bible

## 项目定位

- **项目名**：初入冒险岛
- **类型**：儿童奇幻冒险动画短剧
- **目标形式**：每集 60-120 秒，竖屏或横屏均可适配
- **核心受众**：儿童、家庭观看、亲子共创
- **原作来源**：用户子女原创小说手稿
- **版权状态**：自有原创，版权干净

## 核心创作原则

- 保留孩子原作的想象力，不把故事改成成人套路。
- 可以修顺语句和剧情节奏，但不能擅自改角色关系和核心事件。
- 战斗可以存在，但画面不能血腥、恐怖或成人化。
- 怪物可以有攻击性，但应保留儿童向的理解、善意和成长主题。
- 每集要有一个明确小主题：勇敢、分享、保护、理解、合作。

## 第一季方向

一群孩子进入冒险岛，每个人拥有不同属性魔法。他们通过一关关测试，遇见怪物、谜题和新朋友，逐渐学会合作、勇气和理解别人。

## 当前第一集主题

浩浩带着妹妹沐沐和伙伴们初入冒险岛，遇见因为饥饿而袭击大家的章鱼怪。孩子们先联手打败它，随后发现它受伤又饥饿，沐沐用光魔法治疗它，大家分享食物，通过第一关。
